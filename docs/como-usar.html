Como Usar o Modelo

  1. Clone o repositório: 
    git clone https://github.com/seuusuario/modelo-abnt-ibict.git
cd modelo-abnt-ibict
  2. Compile o projeto com xelatex e biber: 
    xelatex modelo.tex
biber modelo
xelatex modelo.tex
xelatex modelo.tex
  3. Personalize os campos de título, autores, resumos e demais informações no arquivo principal.

Principais Comandos para Personalização

O arquivo cotec.tex disponibiliza comandos que podem ser ajustados pelos autores:

  • \setLogoRevista{caminho} – define o logotipo da revista.
  • \setCheckForUpdatesUrl{url} – configura o link para atualizações do artigo.
  • \secaoNome e \editorNome – identificam a seção e o editor responsável.
  • \dataRecebido, \dataAprovado e \dataPublicado – registram as datas do processo editorial.
  • \pidArtigo e \doiArtigo – configuram os identificadores oficiais do artigo.
  • \tipoLicenca{...} – seleciona a licença de publicação.
  • \textoLicenca, \textoCopyright, \textoCRediT, \textoConflito e \textoFinanciamento – definem os textos das seções obrigatórias.
