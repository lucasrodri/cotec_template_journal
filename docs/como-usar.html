Como Usar o Modelo

  1. Clone o repositório: 
    git clone https://github.com/seuusuario/modelo-abnt-ibict.git
cd modelo-abnt-ibict
  2. Compile o projeto com xelatex e biber: 
    xelatex modelo.tex
biber modelo
xelatex modelo.tex
xelatex modelo.tex
  3. Personalize os campos de título, autores, resumos e demais informações no arquivo principal.
