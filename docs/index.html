Modelo LaTeX ABNT para Revistas Acadêmicas

Este site apresenta um modelo LaTeX desenvolvido para atender às normas da ABNT aplicado ao contexto de revistas acadêmicas e científicas. O objetivo deste modelo é incentivar e facilitar o uso de LaTeX em periódicos brasileiros, oferecendo um template completo, personalizável e alinhado às melhores práticas de editoração.

Aqui você encontrará um resumo das funcionalidades disponíveis, exemplos de utilização e instruções passo a passo para configurar cada elemento do manuscrito. Explore as seções do site e adapte o modelo conforme as necessidades da sua publicação.
